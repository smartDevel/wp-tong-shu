# WP Tong Shu — Chinese Almanac

WordPress Plugin für Tong Shu (Chinese Almanac / 通勝) Kalenderdaten.

## Shortcodes

### Monatskalender
```
[tong_shu_calendar year="2026" month="3"]
```

### Tagesdetail
```
[tong_shu_day year="2026" month="3" day="24"]
```

## Features

- 📅 Monatskalender mit Tong Shu Daten
- 🏮 Himmlische Stämme & Irdische Zweige (天干地支)
- 🐲 Chinesische Tierkreiszeichen
- 🌳 Fünf Elemente (五行)
- ✅/❌ Günstige/Ungünstige Aktivitäten
- 🧭 Glücksrichtungen
- ⚔️ Konflikt-Tierkreiszeichen (冲)
- 🔄 AJAX-Monatsnavigation

## Installation

1. Plugin-Ordner in `/wp-content/plugins/` hochladen
2. Im WP Admin aktivieren
3. Shortcode auf Seite/Beitrag einfügen

## Technisch

- PHP 7.4+
- WordPress 5.8+
- jQuery (WP-bundled)
- GPLv2+
